# StreamWave Frontend

## Overview
StreamWave is a modern video sharing platform with a sleek, responsive design built using React and Chakra UI.

## Features
- User Authentication (Signup/Login)
- Video Upload
- Video Browsing
- Like and Comment Functionality
- Responsive Design

## Tech Stack
- React
- Axios
- React Router
- React Icons

## Prerequisites
- Node.js (v14+)
- npm

## Installation
1. Clone the repository
```bash
git clone https://github.com/zaigham215/streamWave-backend.git
git clone https://github.com/zaigham215/streamWave-frontend.git
```

2. Install dependencies
```bash
npm install
```

## Running the Application
```bash
npm start
```

## Building for Production
```bash
npm run build
```

## Deployment
- Ensure all environment variables are set
- Use your preferred hosting platform (Vercel, Netlify, etc.)

## API Integration
- Backend API: http://localhost:5000
- Authentication via JWT
- Supports video upload, like, comment, and browsing

## Troubleshooting
- Ensure backend server is running
- Check browser console for any errors
- Verify API endpoints and authentication
